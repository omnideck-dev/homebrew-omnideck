# homebrew-omnideck

Official Homebrew tap for the [Omnideck](https://github.com/omnideck-dev/cli) CLI tool.

## Install

```sh
brew tap omnideck-dev/omnideck
brew install omnideck
```

Or in one step:

```sh
brew install omnideck-dev/omnideck/omnideck
```

## Usage

```sh
omnideck --version
omnideck --help
```

## Updating

```sh
brew upgrade omnideck
```
